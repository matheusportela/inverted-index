# Inverted Index

Didactic inverted index

## Requirements

- C++17
- CMake
- GNU Make

## Compiling

```bash
$ ./compile
```

## Downloading data

Download and uncompress 10 WET files containing crawling data:

```bash
$ ./download
```

## Indexing data

Index all WET documents in `data/common-crawl`.

```bash
$ ./run
```

## Searching

Search for single-word terms in inverted index.

```bash
$ ./search
```
